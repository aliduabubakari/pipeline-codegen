# OPOS E2E Suite

Curated OPOS samples used to validate mapping and code generation across all supported targets:

- `airflow_ops_mix.opos.yaml`
- `prefect_ml_lifecycle.opos.yaml`
- `dagster_fanout_quality.opos.yaml`
- `kestra_ingest_notify.opos.yaml`

These inputs are target-agnostic OPOS documents and are generated for all v1 targets (`airflow`, `prefect`, `dagster`, `kestra`) in both `template` and `llm-assisted` modes.
